Viewing Error Details
To view the details of a specific error:
1. Select Error Summary from the sidebar on the left.
2. Locate the error you want to view, and click the link in the Error Text column.
The Error Details screen opens and displays all of the information for that particular error.
At the top of the screen, all of the same information that was shown on the Error Summary is displayed:
•Source system from which the message originated.
•Target system to which the message was sent, or was intended to be sent.
•Error Category of the error (see Error Descriptions for a list of error categories).
•Severity of the error (see Error Statuses and Severity Levels for definitions of severity).
•Error Text describing the error in limited detail.
•Unresolved Count: The number of interface messages that have that error and that are still unresolved.
At the bottom of the screen, there is a Messages table that contains a list of the specific interface messages that had this error, and that are still unresolved. Please note that a given interface message can have more than one error on it, so you might see the same message listed in the Messages table for several different errors. For each message, the following information is displayed:
•Message ID for the message. You can click the link in the Message ID column to view the details for a specific interface message. This opens the Message Details screen (see Viewing Message Details).
•Received Date/Time of the message.
•Patient Name on the message.
•Patient MRN on the message.
