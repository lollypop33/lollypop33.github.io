Editing a System Mapping
If you made a mistake when adding a system mapping, or if you want to edit the name or description of a system mapping, follow these steps:
1. Select System Mapping from the sidebar on the left.
2. Identify the system mapping that you want to edit, and click on the Edit icon in the far left column.
3. In the Edit System Mapping drawer on the right, make the necessary changes to any of the fields, which are described in Step 3 under Adding a System Mapping.
4. Click SAVE.
