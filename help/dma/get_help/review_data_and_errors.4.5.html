Viewing Messages
 
Viewing Message Details
Select the Messages option from the sidebar on the left to view a list of all interface messages, including those with and without errors. You can use this option to investigate any issue that is reported to you, such as interface messages not being processed for a particular patient, interface, or time frame. The report is sorted in reverse chronological order, so that the most recent messages are listed at the top. You can apply filters, or a filter set, to find the specific message(s) that you want to investigate (see Filtering the Data on a Screen).
The Messages option contains a single row for each interface message and displays the following information for each message:
•Message ID of the message. You can click the link in the Message ID column to see the details of a message (see Viewing Message Details).
•Source system from which the message originated.
•Target system to which the message was sent, or was intended to be sent.
•Status of the message (see Message Statuses for definitions of statuses).
•Received Date/Time of the message.
•Patient Name on the message.
•Patient MRN on the message.
