Using this Help System
How to use this help system:
Click this button to open or close the Table of Contents.
Type a word or phrase in the Search field to search for that item in the help system. Use the drop-down filter to the left of the field to search only specific books for the item. Use quotes around a phrase to return results that include all of the words in that phrase (for example, “clinical notes detail” only returns results that include all three of those words, although not necessarily adjacent to each other).
Click Thumbs Up to indicate a topic has been useful or a search result has been successful, or click Thumbs Down to indicate that you have not found what you are looking for.
Click these buttons to go to the Next or Previous sequential topic.
Click this button to return to the Home page.
