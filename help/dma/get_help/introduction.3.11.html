Changing the Sort Order for a Table
Many screens use a table format to display data. Some of these tables have a default sort order, as indicated by an Up Arrow or Down Arrow on one of the column headings.
•Up Arrow means that the report is sorted in ascending order by that column’s data.
•Down Arrow means that the report is sorted in descending order by that column’s data.
You can change the table to sort by a different column’s data, using either of the methods below.
•Method A:
1. Click on a column’s heading name, or on the Up Arrow that appears when you hover your mouse cursor over the heading.
The table is sorted by that column, in ascending order.
2. Once a column is being used to sort the table, you can then toggle between ascending or descending order by clicking repeatedly on either the heading name or on the Up Arrow or Down Arrow .
•Method B:
1. Hover your mouse cursor over the column’s heading to expose the More icon , and then click it.
A context menu opens with a list of options.
2. Choose the Sort by ASC or Sort by DESC option.
The table is sorted by that column, in the order that you selected.
Once you log out, the table reverts to its default sort order.
