Adding a New Code via the Codes Option
You can quickly add new codes via the Codes option.
1. Select Codes from the sidebar on the left.
2. Click + CREATE, located in the top right corner of the screen.
3. In the Create Code drawer on the right, complete the fields for the new code as follows:
-System: Select the system in which this code resides. For example, if it is a code that exist in the source system, select that system here.
-Code: Enter the code.
-Description: Enter a description for the code.
4. Click SAVE.
