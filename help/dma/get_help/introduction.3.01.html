The Data Management Application helps application analysts and integration engineers monitor data integrations between systems, as well as investigate and resolve errors that might occur during the processing of interface messages. For example, your organization might have two different systems for tracking radiology results. In order to make all radiology data available in both systems, an integrations team might configure two-way interface to synchronize the data. The Data Management Application can be used to monitor those interfaces and resolve any errors that occur during the synchronization process.
