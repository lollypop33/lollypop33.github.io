Applying a Saved Filter Set
If you previously saved any filter sets for the current screen, they are listed in the Filter Set drop-down. Filter sets are personal, meaning that only you can see the filter sets that you have created. (See Saving a Filter Set.)
To apply a filter set, click the Filter Set drop-down, located at the bottom left of the screen, and choose the filter set that you want to use. The data on the screen are then filtered based on the criteria defined in the filter set.
To clear the filter set that you have applied, click Clear All.
