Managing Code Mappings
 
Adding a Code Mapping
Editing or Deleting a Code Mapping
Once you have created a system mapping for a code set, you must then enter the mappings for the actual codes in that code set.
EXAMPLE:
You might specify that the test code “IMG123” from Radiology System A should be mapped to test code “GMANDIBLE” in Radiology System B. If there are several codes in Radiology System A that map to the same code in Radiology System B (many-to-one relationship), then you must create multiple mapping entries (for example, map code “IMG123” to “GMANDIBLE” and also “IMG456” to “GMANDIBLE”).
You can also edit or delete a code mapping if a code has been mapped incorrectly.
