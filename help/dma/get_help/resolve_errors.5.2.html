Marking Errors as Ignored
When you have determined that an error can be safely ignored because you have resolved it outside of the Data Management Application, then you can change its status to IGNORED, so that it will be removed from the Error Summary report.
1. Select Error Summary from the slide-out menu drawer.
2. Identify the error that you wish to mark as ignored, and click on the link for that error in the Error Text column.
3. On the Error Details screen, confirm that you have the correct error by reviewing the details at the top of the page, as well as the messages that had that error in the table at the bottom of the page.
4. Click IGNORE ERROR, located at the top right of the screen.
For all of the messages that had this particular error, the error status on the error is changed to IGNORED. Since the Error Summary does not display or aggregate messages with IGNORED errors, the row for this particular error is removed from the Error Summary.
