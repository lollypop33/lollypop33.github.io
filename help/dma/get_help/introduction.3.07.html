Applying Filters
1. Click the FILTERS button, located at the bottom left of the screen, to open the Filter By option.
2. Select your first set of criteria by selecting a Column, an Operator, and a Value.
EXAMPLE:
•Column: Patient MRN
•Operator: contains
•Value: M0001234
NOTE: The text that you enter in the Value field will be applied in a case sensitive manner. For example, “Patient MRN contains M0001234” will return different results from “Patient MRN contains m0001234.”
3. Click Add or Delete to add or remove additional criteria.
4. If you use more than one criteria, make sure the Operator (AND or OR) correctly describes the desired relationship between the criteria. If there are multiple operators, they must all be the same (all ANDs or all ORs).
5. Click APPLY.
The data on the screen are filtered based on your criteria.
To clear the filters that you have applied, click Clear All.
