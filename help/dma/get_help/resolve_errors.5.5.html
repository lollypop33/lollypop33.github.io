Reprocessing Messages from the Error Summary Option
1. Select Error Summary from the slide-out menu drawer.
2. Find the error whose messages you want to reprocess, by either scrolling through the list or using the Filter option (see Filtering the Data on a Screen).
3. Click the link in the Error Text column for the desired error.
4. On the Error Details screen, you can do any of the following:
-Click the check box next to one or more specific messages and then click >>REPROCESS, located at the top left of the table.
-Click the bulk edit check box in the tableâ€™s heading area to select all of the messages on the current page, and then click >>REPROCESS, located at the top left of the table.
NOTE: Only the messages on the current page are reprocessed. For example, if the Pagination area at the bottom of the screen indicates that 1 to 10 of 58 rows are currently displayed, then only rows 1-10 will be reprocessed.
-Click the Message ID link for a specific message, review its details on the Message Details screen, and then click >>REPROCESS MESSAGE, located at the top right of the screen.
If you view the Message Details after a message has finished reprocessing, the Errors table will show errors (if any) from only the latest processing. Prior errors will no longer display.
