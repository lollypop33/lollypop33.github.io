Adding a Code Mapping
The option for adding a new code mapping is accessed via the System Mapping option.
1. Select System Mapping from the sidebar on the left.
2. Identify the system mapping for which you want to add a code mapping, and click on the link in the Name column.
The System Mappings Details screen opens. If any code mappings already exist, they are listed in a Code Mappings table at the bottom of the screen.
3. Click + CREATE, located at the top right of the Code Mappings table, to create a new code mapping.
4. In the Create Code Mapping drawer on the right, enter the following information:
-Source Code: Enter the code that is used in the source system.You can click into the field to view a full list of the source codes that currently exist in the Data Management Application. Or, you can type a few characters of the code to search for it, and then select it from the results list.
If the code does not yet exist in the Data Management Application, a +Create button will display below the field. Click +Create to create a new code on the fly. When the Create Code drawer opens, complete the fields below for the new code and then click SAVE:
◊System: Select the system in which this code resides. For example, if it is a code that exist in the source system, select that system here.
◊Code: Enter the code.
◊Description: Enter a description for the code.
-Target Code: Enter the target code. This should be the equivalent code, in the target system, for the source code you just entered above. This field behaves in a similar manner as the Source Code field -- all of the same methods for selecting or creating a new code are available here as well.
5. Click SAVE.
