Modifying or Deleting a Filter Set
You can change the criteria that is used in a filter set, or delete a filter set, at any time.
1. In the Filter Set drop-down, select the filter set that you want to modify.
2. Click the Gears icon , located to the right of the Filter Set drop-down.
3. The Filter by option opens, displaying the criteria used in the filter set, as well as the filter setâ€™s name.
4. You can now either modify or delete the filter set:
-To modify the filter set: Change any aspect of the filter set, such as changing the attributes of a particular criteria, adding or removing criteria, changing the operator, or changing the name of the filter set. Then click UPDATE to save your changes.
-To delete the filter set: Click DELETE SET.
