Saving a Filter Set
You can save filter criteria that you use often as a filter set, so that you can reuse it at a later time. Filter sets are personal, meaning that only you can see the filter sets that you create. They are also specific to the screen for which they are created.
1. Click the FILTERS button, located at the bottom left of the screen, to open the Filter By option.
2. Select your first set of criteria by selecting a Column, an Operator, and a Value.
EXAMPLE:
•Column: Patient MRN
•Operator: contains
•Value: M0001234
NOTE: The text that you enter in the Value field will be applied in a case sensitive manner. For example, “Patient MRN contains M0001234” will return different results from “Patient MRN contains m0001234.”
3. (Optional) Click Add or Delete to add or remove additional criteria.
4. If you use more than one criteria, make sure the Operator (AND or OR) correctly describes the desired relationship between the criteria. If there are multiple operators, they must all be the same (all ANDs or all ORs).
5. Click Save Filters As.
6. Enter a descriptive name for the filter set in the Save As field.
7. Click SAVE & APPLY.
The data on the screen are filtered based on the criteria defined in the filter set, and the filter set is also added to the Filter Set drop-down, so that you can use it at a later time. (See Applying a Saved Filter Set.)
