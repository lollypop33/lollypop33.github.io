Editing a Code via the Codes Option
You might want to edit a code if either the code or description are incorrect.
1. Select Codes from the sidebar on the left.
2. Find the code you want to edit by either scrolling through the list of codes, or by using the Filters option to search for it (see Filtering the Data on a Screen).
3. Click the Edit icon in the far left column.
4. In the Edit Code drawer on the right, modify the fields for the code as follows:
-System: You cannot change the system for a code.
-Code: Enter the correct code.
-Description: Enter the correct description for the code.
5. Click SAVE.
