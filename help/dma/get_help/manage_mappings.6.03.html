Adding a System Mapping
1. Select System Mapping from the sidebar on the left.
2. Click CREATE, located in the top right corner of the screen.
3. In the Create System Mapping drawer on the right, enter the following information:
-Name: Enter a short but descriptive name for the system mapping. For example, you might enter “Rad Test Codes-System A to System B.”
-Source System: Select the source system, from which the interface messages are derived. In our example, this would be Radiology System A.
-Target System: Select the target system, to which the interface messages are sent. In our example, this would be Radiology System B.
-Description: Enter a full length description of what this system mapping is meant to represent. For example: “Mapping of radiology test codes, from System A to System B.”
4. Click SAVE.
